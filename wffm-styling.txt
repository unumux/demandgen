The following style block needs to be copied into the Header Snippet field of the page item that will be displaying the form.

<style>

form[data-wffm] fieldset + fieldset,
form[data-wffm] input[type=submit].btn,
form[data-wffm] fieldset > legend
 {
  display: none; 
}

form[data-wffm] fieldset:first-of-type table {
  border: none;
}
form[data-wffm] fieldset:first-of-type table tr {
  background-color: #fff;
}

/* Using custom hooks below */

form[data-wffm] .custom-message {
    display: none;
}

</style>
